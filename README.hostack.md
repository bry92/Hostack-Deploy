# Hostack 🚀

**Hostack** is a "deploy-from-GitHub" platform designed for maximum transparency. Think Vercel, but with the engine hood wide open.

## 🌟 Vision
We believe that modern deployment "magic" shouldn't be a black box. Hostack gives you:
- **Transparent Builds:** See the exact Docker images and commands used.
- **Hybrid Detection:** Auto-detects your framework (Next.js, Astro, WP) while giving you full `hostack.yaml` control.
- **CMS Portability:** Manage WordPress as static JSON-based content.
- **Automation-as-Code:** First-class support for `n8n-cli` deployments.

## 🛠️ Included Tools
This repository contains the first set of Hostack building blocks:
- `hostack.yaml`: Declarative deployment configuration.
- `scripts/`: WordPress-to-Static crawling and injection scripts.
- `content/`: Sample JSON-based blog post data.
- `styles/`: Hostack's signature "Transparent Tech" design layer (Tailwind CSS).
